# Introduction

1. Install [Neovim](https://neovim.io/doc/install/).
1. Install [nvm](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating), node>=v20.13.1, and `npm install -g @google/gemini-cli`.
1. Get your Gemini API key from [API Key | Google AI Studio](https://aistudio.google.com/api-keys).
1. `echo 'export GEMINI_API_KEY=[your_gemini_api_key]' >> ~/.bashrc`
1. Place `init.lua` in `~/.config/nvim/`.
1. Open Neovim and run `:Mason`.
1. Install the [Monaco Nerd Font](https://github.com/thep0y/monaco-nerd-font/releases) so Neovim can display icons properly.
